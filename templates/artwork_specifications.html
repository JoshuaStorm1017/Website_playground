Artwork Specifications

Acceptable File Formats

Adobe CC 2018 Illustrator, Photoshop, InDesign

Acceptable file formats are .ai, .eps, .ind

If file is to print “as is”, PDF files are acceptable

It is preferred that print ready files be submitted as a package

Graphics • Resolution • Fonts • Color

  • Vector art is preferred. Be sure to include all graphics (links, support files) used.
  • Be sure all graphics are embedded within the file.
  • Screens, Halftones and Placed images must be at least 150dpi or higher
  • Graphics created for the web are 72dpi – this resolution is poor quality for printing
  • If four color process please use CMYK. If spot colors, use the Pantone Color System (PMS)
  • All fonts must be supplied or created to outlines (paths)

Submitting a File
